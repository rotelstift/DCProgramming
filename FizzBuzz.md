姉：カードの意味を説明するわ。四角くて「おわり」とペアになってるカードは制御用のカード。

姉：そう。プログラムでは上に戻る動きはループ以外でやることは基本的にしないわ。  
　　いろんな歴史的な経緯もあってそうなったのだけど、まぁその理由はプログラムを不必要に複雑にしないように、というものなの。

　　まず、`i % 3 == 0`。これは論理式と呼ばれてるわ。  
　　この論理式は正しい時に`true`、正しくない時に`false`を返すの。  
　　で、この論理式を組み合わせるのが論理演算子ってヤツで、かつを表す`&&`、もしくはを表す`||`、その反対ってのを表す`!`の3種類があるわ。  
弟：へー。組み合わせるってことは、こんな風に書けばいいの？`i % 3 == 0 && i % 5 == 0`
